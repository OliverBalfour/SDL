
# 2D C++/SDL Game

Basic 'game' using C++ and SDL, with SDL's hardware accelerated 2D drawing API using OpenGL.

## Install/build

`git clone [url] game`

`cd game`

`make`

`./main` to run

Or to rebuild binary/levels/gitignore

`make [main/levels/gitignore]`
